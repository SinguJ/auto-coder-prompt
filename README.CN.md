## 介绍

**AutoCoder 提示工程**专注于为 [AutoCoder](https://github.com/SinguJ/auto-coder) 项目研究和优化提示工程技术。

该仓库探索提升提示设计的策略，确保 [AutoCoder](https://github.com/SinguJ/auto-coder) 工具能够更准确、高效地自动生成代码。

## 测试数据集

测试数据集存放在 `/test` 目录下，这些数据主要用于评估提示词的有效性。
